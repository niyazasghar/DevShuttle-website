# DevShuttle | Software Agency

DevShuttle is a premium software agency specializing in high-performance websites, scalable web applications, and mobile applications. Built with a focus on editorial quality, motion sophistication, and technical excellence, this project serves as the digital home for DevShuttle.

## 🚀 Vision
To build elevated digital experiences for ambitious brands, combining strategic thinking with meticulous execution.

## 🛠 Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Typography**: Inter, Plus Jakarta Sans, Montserrat (via Google Fonts)
- **Icons**: Lucide React
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features
- **Modern UI/UX**: Clean, high-contrast design inspired by top-tier creative agencies.
- **Dynamic Animations**: Seamless transitions and entrance animations powered by Framer Motion.
- **Responsive Architecture**: Fully optimized for mobile, tablet, and desktop viewports.
- **SEO Optimized**: Strategic heading hierarchy and metadata configuration.
- **Modular Components**: Reusable, atomic component architecture for scalability.

## 📂 Project Structure
- `src/app/`: Next.js App Router pages and layouts.
- `src/components/`: Reusable React components grouped by feature (layout, home, ui).
- `src/lib/`: Utility functions and shared logic.
- `src/assets/`: Static assets and images.
- `public/`: Publicly accessible static files.

## 🏁 Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm or bun

### Installation
```bash
# Clone the repository
git clone <repository-url>

# Install dependencies
npm install
# or
bun install
```

### Development
```bash
# Run the development server
npm run dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
